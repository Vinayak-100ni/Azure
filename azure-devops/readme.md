### Azure Boards
Work tracking system to manage tasks, bugs, sprints, and agile workflows.

### Azure Repos
Git-based version control system to manage source code repositories.

### Azure Pipelines
CI/CD service to build, test, and deploy applications automatically.

### Azure Test Plans
Tool for manual and exploratory testing with test case management.

### Azure Artifacts
Package management system to store and share dependencies (npm, Maven, NuGet, etc.).
